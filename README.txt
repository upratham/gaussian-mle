# Maximum Likelihood Estimation with Gaussian Distribution

This repository demonstrates how to perform **Maximum Likelihood Estimation (MLE)** using Gaussian (Normal) distributions. 
The project includes both data and a Jupyter Notebook with implementation, visualizations, and explanations.
